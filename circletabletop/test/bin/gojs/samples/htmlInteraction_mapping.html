Selecting nodes in the main Diagram will display information here

TEXT prefix legend:

"" (no prefix) - Writes text as label into location definition overwrites

"~" - Writes text as label directly into LocationDefinition.
"#" - Load content by LocationDefinitionID. Cancels any current location definition overwrites.
"*" - Save out current overwrites over new LocationDefinition using text as both label and LocationDefinitionID.
"*#" - Save out current overwrites over new LocationDefinition, using text as LocationDefinitionID
NOTE: When saving, if text is an empty string, will save all existing overwrites over current LocationDefinitionID.
